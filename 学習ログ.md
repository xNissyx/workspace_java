# 2026/6/10
実施内容
- 配列

Cとの違い
- 書き方：int[] a = new int[3];
- Javaは配列外参照をゆるさない（コンパイルは通るが、実行時にArrayIndexOutOfBoundsExceptionがでる）
- オブジェクトなので、メモリに格納されているのは参照先のアドレス

気づき
- lengthは配列のフィールド（メンバ変数）なのでメソッドではない
- Stringはlengthメソッドだが、配列はlengthフィールド
